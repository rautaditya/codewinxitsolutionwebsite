# Public Assets

Place the following files here:

- `logo.png` — Your brand logo (recommended: 200x60px PNG with transparent background)
- `favicon.ico` — Browser tab icon (16x16 or 32x32px)
- `images/hero.png` — Hero section background or illustration
- `images/og-image.jpg` — Open Graph social share image (1200x630px)
